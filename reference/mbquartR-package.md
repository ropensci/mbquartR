# Finding Manitoba, Canada, quarter sections made easier

`mbquartR` is an R package for locating quarter sections in Manitoba
using lat/long coordinates or the legal land descriptions using the
Manitoba Original Survey Legal Descriptions data set from the DataMB
website (<https://geoportal.gov.mb.ca/>).

## Details

There are four main functions in this package:

1.  Download the Manitoba Original Survey Legal Descriptions data set

- [`quarters_dl()`](https://docs.ropensci.org/mbquartR/reference/quarters_dl.md)

1.  Search for legal land descriptions using lat and long coordinates

- [`search_coord()`](https://docs.ropensci.org/mbquartR/reference/search_coord.md)

1.  Search for lat and long coordinates using legal land descriptions

- [`search_legal()`](https://docs.ropensci.org/mbquartR/reference/search_legal.md)

1.  Plot search results on an interactive map

- [`map_quarter()`](https://docs.ropensci.org/mbquartR/reference/map_quarter.md)
